Prédiction d'une Intelligence Artificielle

L'espérance de vie des patients atteints du SIDA est sans cesse en train d'augmenter.
Nous avons entraîné une modèle d'IA afin d'essayer de prédire cette espérance de vie pour les années à venir

Nous avons utilisé le jeu de données open-source suivant : Lien Kaggle.
Il a été proposé pour la première fois par M Roser et al.

L'ensemble de données fournit l'espérance de vie du VIH SIDA changeant par rapport à l'année. Ces informations sont fournies par pays.
Les échantillons de cet ensemble de données peuvent être représentés sous forme de séries chronologiques.
Pour ce dernier, un modèle de prévision peut être appris sur le jeu de données afin de prédire dans le futur l'espérance de vie des patients VIH SIDA.

De nombreux modèles de prévision peuvent être utilisés, dans ce cas un réseau de neurones LSTM a été utilisé pour résoudre cette tâche.
Le réseau de neurones se compose d'une couche LSTM avec 3 unités suivies d'une couche entièrement connectée fournissant la prédiction d'un horodatage futur.

Le modèle a été implémenté dans Tensorflow Keras. Les séries chronologiques d'entrée ont été normalisées afin d'avoir une moyenne de 0 et un écart type de 1.
Le modèle a été formé en utilisant une taille de lot de 32 pour 500 époques à l'aide d'un optimiseur Adam et d'une perte d'erreur quadratique moyenne.

Pour la phase de prédiction, 20 horodatages ont été prédits. Chaque horodatage prédit est utilisé pour prédire le suivant.
Les résultats ont montré que le modèle a appris différents modèles et a pu prédire qu'à l'avenir, certains pays auront la même espérance de vie et certains auront en fait un âge d'espérance de vie plus élevé.
